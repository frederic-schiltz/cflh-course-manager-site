CFLH Course Manager

A desktop application for managing language course administration.

CFLH Course Manager is designed to support authorized staff in handling course participant data and communication tasks efficiently.

Main features

  • Manage participant and course-related data
  • Manage course documents such as certificates
  • Send emails with attachments to course participants

Use of Gmail

The application connects to Gmail only for sending emails explicitly initiated by the user. It does not read inbox contents or analyze mailbox data.
