Privacy Policy

CFLH Course Manager

This Privacy Policy explains how CFLH Course Manager handles data.

1. Purpose of the application

CFLH Course Manager is a desktop application used to manage course-related administrative data and to send emails to course participants.

2. Data processed by the application

The application may process participant names, contact details, course-related administrative data, and generated course documents such as certificates.

All such data is processed locally on the user’s computer.

3. Use of Google services

The application uses the Gmail API solely to send emails initiated by the user.

Scope used:
https://www.googleapis.com/auth/gmail.send

The application does not:

  • read emails
  • access inbox contents
  • modify existing messages
  • store Gmail data on external servers

4. Data storage

The application does not store personal data on external servers. Data remains on the user’s local system unless explicitly sent by email by the user.

5. Data sharing

No personal data is shared with third parties, except when emails are sent by the user through Gmail.

6. Security

The application uses Google OAuth 2.0 authentication for Gmail access. No Gmail password is stored by the application.
